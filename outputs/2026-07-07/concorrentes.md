# Análise Competitiva — Máxima Finance IA
*07/07/2026 | Turno da tarde | Bella*
*Fontes: pesquisa web (julho 2026)*

---

## RESUMO EXECUTIVO

Após pesquisa atualizada, o gap estratégico da Máxima Finance permanece **intacto e crescente**: nenhum dos três principais concorrentes analisados tem IA que interpreta DRE em linguagem simples para o dono da empresa. Omie continua sendo a maior ameaça — mas ainda está no eixo operacional (ERP, NF, caixa via WhatsApp), não no diagnóstico financeiro.

---

## 1. TREASY
**Posicionamento:** Planejamento orçamentário / FP&A para PMEs

### O que está fazendo em 2026
- Lançou **Kit de Planejamento Orçamentário 2026**: planilha de DRE + simulação de cenários + metodologia proprietária
- Está aprofundando foco em **FP&A para PMEs** como diferencial competitivo
- Produz conteúdo educativo pesado (blog, podcasts Controller Cast) — estratégia de topo de funil forte
- Publicou análise sobre desempenho econômico das PMEs em 2025 como gancho para 2026
- Está aproveitando a **Reforma Tributária (CBS/IBS)** como pauta para atrair planejadores financeiros

### Fraquezas
- Produto voltado para **controllers e CFOs**, não para o dono de PME sem background financeiro
- Não tem IA conversacional — é planilha e dashboard estático
- Interface complexa para quem não domina FP&A
- Não interpreta DRE — você preenche, ele processa. Não diagnostica.

### Nível de ameaça para Máxima Finance
🟡 **Baixa** — público diferente (controlador vs. dono), não tem IA conversacional, não compete no mesmo problema

---

## 2. CONTA AZUL
**Posicionamento:** ERP / Gestão financeira para PMEs + contadores

### O que está fazendo em 2026
- Lançou **Conta AI Captura**: IA (modelo Google DeepMind/Gemini) que lê e lança documentos automaticamente — boletos, NFs, extratos, faturas
- Funcionalidade de **geração de relatórios via linguagem natural**: "me dê o relatório de vendas de junho" → IA gera
- Integração via ChatGPT/OpenAI disponível via plataforma Albato
- Confirmado: evento **Conta Azul CON 2026** em São Paulo (5–6/agosto) — alinhado com contexto da Karina
- Está acelerando o uso de IA para **automatizar tarefas manuais do contador**, não para interpretar dados para o empresário

### Fraquezas
- IA focada em **automação operacional** (lançamentos, categorização) — não em diagnóstico estratégico
- O produto é para **escritórios contábeis e empresas com contador ativo**, não para o empresário autônomo
- Relatório gerado por linguagem natural ainda é descritivo — não diz "o que fazer"
- Base enorme (custo de migração alto), mas proposta de valor não compete diretamente

### Nível de ameaça para Máxima Finance
🟠 **Média** — tem IA, tem base enorme, está investindo pesado. Mas o gap persiste: Conta Azul automatiza o trabalho do contador. Máxima Finance fala direto com o dono.

---

## 3. OMIE
**Posicionamento:** ERP completo para PMEs — e agora via WhatsApp

### O que está fazendo em 2026
- **ERP 100% operável via WhatsApp** (texto + áudio): emissão de NF, controle de estoque, boletos, movimentações bancárias, fluxo de caixa
- Comandos via WhatsApp como: *"Qual o saldo da minha conta no Banco do Brasil?"* / *"Qual o fluxo de caixa hoje?"*
- Desenvolvido por ~2 anos, testado em beta com ~1.000 clientes, agora em escala
- Meta declarada: **1 milhão de clientes até 2030** + se tornar empresa de US$1 bilhão
- Próximo passo previsto: **crédito integrado via WhatsApp**
- IA integrada desde 2019 — não é novidade, é camada estrutural

### Fraquezas
- Ainda focado em **operação** (emitir NF, consultar saldo, movimentar estoque) — não em diagnóstico
- Não interpreta DRE nem entrega diagnóstico financeiro estratégico
- Não tem chat financeiro educativo (como a Sofia) — é assistente operacional
- Escopo de crédito integrado ainda é promessa, não produto lançado

### Nível de ameaça para Máxima Finance
🔴 **Alta e crescente** — não compete hoje, mas tem a infraestrutura (WhatsApp, IA, base de clientes, capital) para expandir para diagnóstico financeiro. **Urgência de lançar antes que Omie amplie escopo é real.**

## CONCLUSÃO ESTRATÉGICA

**O gap é real. A janela está aberta — mas com prazo.**

Conta Azul e Treasy não ameaçam o posicionamento de diagnóstico financeiro com IA conversacional. Omie é o único que pode chegar lá — e tem capital, base e estrutura para isso.

**Recomendação:** deploy antes de agosto. Conta Azul CON (5–6/ago) vai gerar atenção do mercado para soluções financeiras com IA. Estar no ar nessa janela é diferencial de PR e captação.
