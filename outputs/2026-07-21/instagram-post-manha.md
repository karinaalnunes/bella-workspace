## LEGENDA

Você paga aluguel todo mês.
Paga salários. Paga luz, contador, sistema.

Mesmo quando a loja está vazia.
Mesmo quando o time está no ócio.
Mesmo quando não entra um real.

Isso tem nome: **custo fixo.**
E ele não espera.

O problema não é ter custo fixo — toda empresa tem.
O problema é não saber **qual é o mínimo que você precisa faturar** para esses custos não comerem o seu lucro.

Esse número é o **ponto de equilíbrio.**

E ele está escondido no seu DRE.

👉 Soma todas as suas despesas fixas do mês.
👉 Divide pela sua margem de contribuição (quanto sobra de cada R$100 vendido depois dos custos variáveis).
👉 O resultado é o faturamento mínimo para você não entrar no vermelho.

Abaixo disso → prejuízo silencioso.
Acima disso → cada real a mais vai para o seu bolso.

Simples assim. Mas poucos empresários fazem essa conta.

Você faz?
