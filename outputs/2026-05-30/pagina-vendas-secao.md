## PARTE 3 — MOMENTOS DE ENGAJAMENTO (Durante o webinar)

*Distribua ao longo da apresentação para manter atenção e preparar o terreno para o CTA.*

### Momento 1 — Poll (~20 min de webinar)
**Pergunta para o chat:**
> "Me conta: qual é o seu maior desafio com as finanças do negócio hoje?
> A — Não entendo os números do DRE
> B — Sei os números mas não sei o que fazer com eles
> C — Não tenho tempo de olhar o financeiro
> D — Meu contador não me explica nada"

*Leia os resultados, faça comentários curtos sobre cada opção, conecte com o conteúdo que vem em seguida.*

### Momento 2 — Exercício ao vivo (~35 min de webinar)
> "Agora quero que vocês façam isso junto comigo. Pega o celular ou um papel. Escreve o faturamento bruto do mês passado. Agora escreve os custos variáveis — o CMV, os impostos sobre a venda. Subtrai. Esse é o seu resultado bruto. Quantos de vocês nunca tinham calculado isso antes? Escreve no chat."

*Pausa, leia respostas, reforce que isso é exatamente o que a Sofia faz automaticamente.*

### Momento 3 — Pergunta de dor (~50 min de webinar)
> "Uma última pergunta antes de eu mostrar a solução: imagina que você pudesse fazer uma pergunta pro seu DRE e ele te respondesse em português simples, sem precisar de contador, agora. Qual seria essa pergunta? Manda no chat."

*Leia 3–4 perguntas em voz alta. Responda brevemente cada uma. Isso cria o gancho perfeito para apresentar o Chat Sofia.*

## PARTE 6 — TOKENS DEV

```typescript
// Evento: webinar encerrado → disparar automações
type WebinarEndEvent = {
  webinarId: string;
  attendees: string[]; // IDs Supabase dos que ficaram > 30 min
  noShows: string[];   // IDs que se inscreveram mas não compareceram
  timestamp: string;   // ISO 8601
}

// Supabase: atualizar tabela waitlist após webinar
await supabase
  .from('waitlist')
  .update({ attended_webinar: true, attended_at: new Date().toISOString() })
  .in('id', attendees);

// Resend: disparar replay para quem assistiu
await resend.emails.send({
  from: 'Karina <karina@maximafinance.com.br>',
  to: attendeeEmails,
  subject: '[REPLAY] O DRE que você nunca viu — acesso por 48h',
  react: ReplayEmailTemplate({ checkoutUrl: process.env.CHECKOUT_URL }),
});

// Resend: disparar sequência No-Show (arquivo 05-26)
// Primeiro e-mail: 2h após fim do webinar
await resend.emails.send({
  from: 'Karina <karina@maximafinance.com.br>',
  to: noShowEmails,
  subject: 'Você perdeu — mas tem replay',
  react: NoShowEmail1Template(),
});
```
