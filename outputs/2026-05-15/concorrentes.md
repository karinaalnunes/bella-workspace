# Análise Competitiva v26 — Máxima Finance IA
*Turno da tarde — 15/05/2026*

---

## RESUMO EXECUTIVO

Gap estratégico da Máxima **permanece intacto**: nenhum dos três concorrentes monitorados entrega diagnóstico de DRE em linguagem simples para o empresário. A janela de lançamento estimada é de **3–5 meses** antes que Omie expanda o escopo do agente generativo para análise financeira.

---

## 1. TREASY

**Posicionamento:** FP&A (Financial Planning & Analysis) — planejamento orçamentário e controladoria

### Novidades detectadas (maio/2026)
- Nenhum lançamento de IA para PME identificado neste ciclo
- Foco continua 100% enterprise: CFOs, controllers, grandes operações
- FP&A Summit (12–13/mai) confirmou posicionamento: palestrantes e cases de médias/grandes empresas
- Integra com ERPs (TOTVS, Omie, Conta Azul, Nibo) como ferramenta complementar, não concorrente direto deles
- Pricing público no Capterra/GetApp: posiciona produto como premium, fora do ticket de PME

### Análise estratégica
| Critério | Treasy | Máxima Finance IA |
|---|---|---|
| Público-alvo | Controllers / CFOs de médias-grandes | Donos de PME (35–55 anos) |
| Interface | Técnica, planilhão | Conversa em linguagem simples |
| IA conversacional | ❌ Não tem | ✅ Chat Sofia |
| Leitura de DRE para leigos | ❌ Não tem | ✅ Core da solução |
| Ticket | Alto (enterprise) | R$97–R$297/mês |

**Ameaça:** Baixa. Mercados diferentes. O Treasy é parceiro potencial (integração), não concorrente.

**Oportunidade:** Empresários que "saem do Treasy" por ser caro demais ou complexo demais são leads qualificados para a Máxima.

## 2. CONTA AZUL

**Posicionamento:** ERP generalista para PMEs + plataforma de serviços para contadores/BPOs

### Novidades detectadas (maio/2026)
- **Conta AI Captura** — integração com **grupos de WhatsApp** agora permite multi-CNPJ: o mesmo número envia documentos para diferentes contas (foco BPO/escritórios contábeis que gerenciam vários clientes)
- **Funcionalidade:** captura documentos (boletos, notas, comprovantes) → sugere lançamento contábil → não interpreta resultado financeiro
- **Evento:** Conta Azul CON 2026 confirmado para **5–6 de agosto**, Transamérica Expo Center, São Paulo. Temas: IA, reforma tributária, contabilidade digital, futuro do BPO financeiro
- Visão declarada: usar IA para transformar contador em "consultor financeiro" — foco no canal parceiro, não no empresário direto
- Integração com ChatGPT/OpenAI disponível via plataforma Albato (terceiros), não nativa

### Análise estratégica
| Critério | Conta Azul | Máxima Finance IA |
|---|---|---|
| IA para quem? | Para o contador (B2B2C) | Para o empresário direto (B2B) |
| Interpreta DRE? | ❌ Não — só captura documentos | ✅ Sim — diagnóstico em linguagem simples |
| Canal principal | Contador / BPO | Dono de negócio |
| WhatsApp | Captura doc via WhatsApp | — |
| Ticket | ERP + planos escalonados | R$97–R$297/mês |

**Ameaça:** Média. Tem base enorme de clientes PME, mas a IA deles serve ao contador, não ao empresário. O gap permanece.

**Alerta para agosto:** O CON 2026 (5–6/ago) pode ser palco de anúncio de IA voltada ao empresário. **Deploy da Máxima deve acontecer antes dessa data.**

**Oportunidade:** Usuários do Conta Azul que querem entender o resultado financeiro — não só lançar documentos — são leads naturais.

## 3. OMIE

**Posicionamento:** ERP completo para PMEs + WhatsApp + IA agente em desenvolvimento

### Novidades detectadas (maio/2026)
- **WhatsApp ERP:** ferramenta em produção (após ~2 anos de desenvolvimento e beta com 1.000 clientes). Todo o ERP operável via WhatsApp — texto e áudio. Desenvolvido sobre AWS + Meta APIs oficiais
- **Agente generativo declarado como roadmap:** Omie planeja transformar o WhatsApp ERP em agente conversacional e generativo — "todo o poder de gestão a uma mensagem de distância"
- **Dois focos de IA para 2026:**
  1. Reduzir atrito na adoção de software de gestão
  2. IA como aliada estratégica — gerar insights, navegar complexidade fiscal
- **Aporte:** Partners Group adquiriu 15% da Omie por US$ 100 milhões (confirmado) → capacidade de aceleração de produto aumentada significativamente
- **App BI Omie:** dashboards de decisão estratégica para PMEs — análise de tendências, indicadores de performance

### Análise estratégica
| Critério | Omie | Máxima Finance IA |
|---|---|---|
| WhatsApp | ✅ ERP completo operacional | ❌ (não é o foco) |
| Agente generativo | 🔄 Roadmap (em desenvolvimento) | ❌ |
| Interpreta DRE? | ❌ Ainda não — foco em operação ERP | ✅ Core da solução |
| Chat financeiro em português simples | ❌ Não tem | ✅ Chat Sofia |
| Ticket | ERP (~R$200–R$500+/mês) | R$97–R$297/mês |
| Escopo | ERP completo (NF, estoque, etc.) | Diagnóstico financeiro especializado |

**Ameaça:** Alta e crescente. O Omie é o concorrente mais perigoso.
- Tem base: milhares de PMEs já na plataforma
- Tem capital: US$ 100M para acelerar desenvolvimento
- Tem o canal: WhatsApp já operacional
- Falta: interpretação de DRE em linguagem simples + chat financeiro como o Sofia

**Janela estimada:** 3–5 meses até o agente generativo do Omie expandir escopo para análise financeira. Cada semana sem lançar é vantagem entregue.
