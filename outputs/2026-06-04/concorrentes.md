## 1. TREASY

**Posição:** FP&A e planejamento orçamentário para empresas

### Novidades 2026
- **FP&A Summit (12–13/mai/2026 — São Paulo):** evento presencial com +20 palestrantes e +1.000 líderes financeiros. Foco em frameworks de decisão, governança e cenários — audiência é CFO, controller, gestor financeiro sênior. PME não é o público.
- **Kit de Planejamento Orçamentário 2026:** material gratuito para construção de orçamento empresarial.
- **"Jornada da Meta ao Resultado 2026":** trilha paga — orçamento completo da empresa em 4 dias.
- **FP&A para PMEs (conteúdo):** blog post posicionando FP&A como acessível a qualquer empresa — mas sem produto específico para PME. É conteúdo, não solução.

### Análise para Máxima Finance
| Dimensão | Treasy | Máxima Finance |
|---|---|---|
| Público-alvo | CFO / Controller / enterprise | Dono de PME sem equipe financeira |
| Interface | Planilhas estruturadas / dashboards complexos | Upload DRE → diagnóstico automático |
| IA | Não há | Claude API — interpretação em linguagem natural |
| Chat financeiro | Não | Sofia (dúvidas em tempo real) |
| Barreira de entrada | Alta — exige conhecimento de FP&A | Baixa — qualquer empresário usa |

**Nível de ameaça:** 🟡 BAIXO — não disputa o mesmo comprador. O evento Summit confirma foco enterprise.

**Gap confirmado:** Treasy fala FP&A; Máxima fala DRE para o dono que não tem controller.

## 2. CONTA AZUL

**Posição:** ERP + contabilidade para PMEs — maior base de clientes do segmento

### Novidades 2026
- **Conta AI:** guarda-chuva de IA generativa lançado em 2025/2026. Parceria com Google DeepMind (Gemini).
  - **Conta AI Captura:** captura automática de documentos (boletos, NFs, comprovantes) via imagem/PDF. Classifica categorias contábeis com base no histórico. **Não interpreta DRE.**
  - **Integração WhatsApp multi-CNPJ:** o número de WhatsApp pode enviar documentos para múltiplas contas — recurso voltado para BPOs e contadores que gerenciam vários clientes.
  - **Relatórios com IA:** usuário pede em linguagem simples → IA gera sugestão de relatório. Ainda em evolução.
- **Conta Azul CON 2026:** maior conferência de contabilidade do país, prevista para **5–6 de agosto de 2026, São Paulo.** Potencial palco para anúncio de novidades de IA.

### Análise para Máxima Finance
| Dimensão | Conta Azul | Máxima Finance |
|---|---|---|
| Público-alvo | PME + contador | Dono de PME sem contador (ou que não entende o contador) |
| IA atual | Captura de documento — não interpreta DRE | Análise + diagnóstico do DRE em linguagem simples |
| Chat financeiro | Não — IA é back-office | Sofia — conversa sobre finanças do negócio |
| Preço entrada | Planos a partir de ~R$199/mês | R$97/mês (Pro) |
| Curva de uso | Alta — sistema completo | Baixa — começa só com o DRE |

**Nível de ameaça:** 🔴 ALTA — maior base de clientes PME do Brasil. A Conta AI ainda não interpreta DRE, mas está evoluindo rápido. O CON 2026 em agosto é o maior risco de anúncio de produto concorrente direto.

**Gap confirmado:** Conta Azul automatiza tarefas contábeis. Máxima interpreta o resultado e diz o que fazer.

## 3. OMIE

**Posição:** ERP conversacional via WhatsApp — primeiro do mundo nesse formato

### Novidades 2026
- **ERP 100% operável via WhatsApp (produção):** emissão de NF, controle de estoque, movimentações financeiras, consulta de indicadores — tudo por texto ou voz no WhatsApp. Em produção desde 2025/2026.
- **Crédito via WhatsApp (roadmap confirmado):** contratação de crédito conversacional por voz ou texto — próximo passo declarado publicamente.
- **IA para compliance tributário:** classificação fiscal automatizada, antecipação de falhas operacionais com risco de multa, análise de dados tributários em tempo real.
- **Onboarding e treinamento via WhatsApp:** meta declarada de mover todo o ciclo do cliente (contratação → uso → suporte) para o WhatsApp.
- **Omie Academy — Reforma Tributária 2026:** capacitação de profissionais para os impactos da reforma.

### Análise para Máxima Finance
| Dimensão | Omie | Máxima Finance |
|---|---|---|
| Diferencial | ERP conversacional via WhatsApp | Diagnóstico de DRE em português simples |
| IA atual | Operacional (NF, estoque, cobrança) | Interpretativa (DRE → diagnóstico → decisão) |
| Chat financeiro | Consulta de indicadores via WhatsApp | Sofia — análise estratégica do DRE |
| Interpreta DRE? | ❌ Não | ✅ Core do produto |
| Receita estimada | US$100M+ (Série D confirmada) | Early-stage / pré-lançamento |

**Nível de ameaça:** 🔴 CRÍTICA — maior ameaça competitiva. O Omie está expandindo escopo (operacional → estratégico → crédito). Se adicionar interpretação de DRE ao chat do WhatsApp, o gap da Máxima fecha. Janela estimada: **3–4 meses**.

**Gap confirmado (ainda):** Omie faz O QUÊ aconteceu (lança NF, paga boleto, consulta estoque). Máxima explica O QUÊ SIGNIFICA (margem, ponto de equilíbrio, onde o dinheiro foi). Diferente — mas a janela está fechando.

## RESUMO EXECUTIVO

| Concorrente | Ameaça | Diferenciador da Máxima vs. ele |
|---|---|---|
| **Treasy** | 🟡 Baixa | Foco enterprise; Máxima é para PME sem controller |
| **Conta Azul** | 🔴 Alta | IA só captura docs; Máxima interpreta DRE e diz o que fazer |
| **Omie** | 🔴 Crítica | ERP operacional; Máxima diagnóstica — mas janela está fechando |

### Janela de lançamento
- **Omie:** 3–4 meses antes de expandir escopo para diagnóstico
- **Conta Azul CON:** 5–6 de agosto de 2026 — risco de anúncio de IA para PME
- **Conclusão:** lançar o MVP antes de agosto é prioridade estratégica

### Gap que persiste (nenhum concorrente atende)
> Nenhum dos três converte o DRE em diagnóstico conversacional para o empresário que não sabe finanças. Esse é o único diferenciador que a Máxima precisa defender — e rápido.
