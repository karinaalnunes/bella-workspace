## NOTAS DE IMPLEMENTAÇÃO (para o dev)

### Integração Stripe — IDs dos produtos a criar:
- `prod_gratuito` → price: R$ 0 (free tier)
- `prod_pro_mensal` → price: R$ 97/mês (recorrente)
- `prod_business_mensal` → price: R$ 297/mês (recorrente)

### Variáveis de ambiente necessárias:
```
STRIPE_PUBLIC_KEY=pk_live_...
STRIPE_SECRET_KEY=sk_live_...
STRIPE_WEBHOOK_SECRET=whsec_...
STRIPE_PRICE_PRO=price_...
STRIPE_PRICE_BUSINESS=price_...
```

### Lógica de checkout:
- Gratuito → criar conta diretamente no Supabase, sem Stripe
- Pro e Business → redirecionar para Stripe Checkout → webhook confirma → Supabase atualiza `plan_type`

### Highlight visual recomendado:
- Plano Pro: borda colorida + badge "Mais Popular"
- Plano Business: badge "Melhor Custo-Benefício para Grupos"
- Gratuito: visual mais sóbrio, sem destaque
