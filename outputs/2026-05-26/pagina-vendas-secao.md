## TOKENS DEV (Supabase + Resend)

```typescript
// Trigger: 2h após fim do webinar para inscritos com attended = false
// Query Supabase:
const noShows = await supabase
  .from('webinar_registrations')
  .select('*')
  .eq('webinar_id', webinarId)
  .eq('attended', false)

// Disparar sequência Resend:
await resend.emails.send({
  from: 'Karina <karina@maximafinance.com.br>',
  to: lead.email,
  subject: 'Você perdeu o webinar — gravei um recado pra você',
  react: EmailNoShow1({ name: lead.name, replayUrl: REPLAY_URL })
})

// Agendar próximos e-mails via Resend Schedules ou Supabase Edge Function com pg_cron:
// D+2: 'O que rolou no webinar'
// D+3: 'Só mais 24h'
// D+4: 'Última chance'

// WhatsApp D+1 e D+4 via Evolution API ou Z-API:
// endpoint: /message/sendText/{instance}
// payload: { number: lead.whatsapp, text: MENSAGEM }

// Marcar como "no_show_sequence_started = true" para evitar duplicata:
await supabase
  .from('webinar_registrations')
  .update({ no_show_sequence_started: true })
  .eq('id', lead.id)
```
