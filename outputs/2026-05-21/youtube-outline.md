## BLOCO 1 — O QUE É O DRE MENSAL (1:30 – 4:00)

**Roteiro:**
```
O DRE mensal mostra o resultado de um único período:
receitas, custos, despesas e lucro de janeiro, fevereiro, março...
cada mês separado.

É o que a maioria das empresas recebe do contador.
E é útil — mas incompleto sozinho.

Por que? Porque o negócio tem sazonalidade.

Exemplo prático:
→ Ótica: dezembro e janeiro são os meses mais fortes
→ Construtora: fevereiro e março costumam ser mais fracos
→ Loja de material escolar: janeiro e julho explodem

Se você olha só fevereiro de uma ótica e vê resultado negativo,
pode entrar em pânico. Mas se olhar o acumulado, vai ver que
o ano inteiro está indo bem.

O contrário também é verdade — e mais perigoso.
```

## BLOCO 2 — O QUE É O DRE ACUMULADO (4:00 – 7:00)

**Roteiro:**
```
O DRE acumulado soma todos os meses do ano até o período atual.

Se você está em maio, o DRE acumulado mostra:
→ Receita total de janeiro a maio
→ Custos totais de janeiro a maio
→ Lucro ou prejuízo real do que foi de 2026 até agora

Isso é o que responde perguntas como:
"Minha empresa está crescendo ou só teve um mês bom?"
"Estou num ritmo de meta ou estou atrás?"
"O ano vai fechar positivo se continuar assim?"

O DRE mensal não responde nenhuma dessas.
```

## BLOCO 3 — O ERRO CLÁSSICO (7:00 – 10:30)

**Roteiro:**
```
Deixa eu te contar um caso que acontece com frequência.

Empresário de serviços, fatura R$ 120 mil/mês em média.

Março foi incrível: R$ 180 mil.
Margem líquida de 18%.
Ele contrata um funcionário, compra equipamento novo
e decide dar um pró-labore maior.

Aí chega maio.
Olha o acumulado de janeiro a maio:
→ Receita total: R$ 580 mil
→ Despesas fixas aumentaram com o novo funcionário e o pró-labore
→ Resultado líquido acumulado: -R$ 12 mil

Março foi uma exceção, não uma tendência.
Mas ele tomou decisões como se fosse tendência.

O DRE acumulado teria mostrado isso antes.

Como identificar esse erro no seu negócio:
1. Pegue o DRE dos últimos 5 meses
2. Some as receitas — depois some os custos e despesas
3. Calcule: resultado líquido acumulado ÷ meses = resultado médio mensal real

Se o resultado médio real for muito diferente do mês que você considera "normal",
você tem um problema de leitura de dados — não necessariamente de negócio.
```

## BLOCO 4 — COMO USAR OS DOIS JUNTOS (10:30 – 13:30)

**Roteiro:**
```
A resposta não é abandonar o DRE mensal.
É usar os dois com finalidades diferentes.

DRE mensal → decisões operacionais do próximo mês
→ Vou fechar uma venda grande ou economizar esse mês?
→ Preciso renegociar algum fornecedor?
→ Qual produto ou serviço puxou o resultado?

DRE acumulado → decisões estratégicas
→ Posso contratar?
→ Consigo honrar um financiamento?
→ Estou no ritmo para bater a meta do ano?
→ A empresa está crescendo de verdade ou oscilando?

Prática recomendada:
→ Toda semana: olhar caixa (não é DRE, é fluxo de caixa)
→ Todo mês: analisar o DRE mensal com o contador
→ A cada trimestre: revisar o DRE acumulado e comparar com a meta do ano

Esse ritmo de três camadas é o que separa o empresário
que reage do empresário que decide.
```
