## TOKENS PARA DEV / CONFIGURAÇÃO

### UTM Parameters (padrão para todos os links):
```
Search Grupo 1.1: ?utm_source=google&utm_medium=cpc&utm_campaign=search_dre_ia&utm_content=grupo_dre_ia
Search Grupo 1.2: ?utm_source=google&utm_medium=cpc&utm_campaign=search_gestao_pme&utm_content=grupo_gestao_pme
Search Grupo 1.3: ?utm_source=google&utm_medium=cpc&utm_campaign=search_concorrencia&utm_content=grupo_concorrencia
Display Remarketing: ?utm_source=google&utm_medium=display&utm_campaign=remarketing_site&utm_content=banner_{tamanho}
```

### Conversões a configurar no Google Ads:
```javascript
// Evento 1: Cadastro /diagnostico-gratuito
gtag('event', 'conversion', {
  'send_to': 'AW-XXXXXXXXX/YYYYYYY',
  'event_category': 'lead',
  'event_label': 'diagnostico_gratuito'
});

// Evento 2: Upgrade para plano pago
gtag('event', 'conversion', {
  'send_to': 'AW-XXXXXXXXX/ZZZZZZZ',
  'value': 97.00,
  'currency': 'BRL',
  'event_label': 'upgrade_pro'
});
```

### Audiences para criar no Google Analytics 4 (para remarketing):
```
1. "Visitou home + não converteu" — condição: page_view(/) AND NOT conversion_event
2. "Visitou /diagnostico-gratuito" — condição: page_view(/diagnostico-gratuito)
3. "Usuário free inativo 15d" — via Supabase → GA4 audience upload
```
